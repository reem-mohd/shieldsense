# ShieldSense
A gamified web-based social engineering awareness training platform built for UAE university students.

## What it does
Trains users to recognize social engineering tactics through 8 scenario-based challenges with pre/post knowledge assessment.

## Tech used
HTML5, CSS3, JavaScript ES6+, IndexedDB, PBKDF2-SHA256

## Key features
- STRIDE threat modelling
- Secure client-side storage (310,000 iteration PBKDF2, salted)
- Pre/post knowledge check to measure learning impact
